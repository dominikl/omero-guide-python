Simple FRAP script - Step by Step
=================================

**Description:**
----------------

The script does a simple FRAP analysis using an ellipse ROI previously
saved on an image, creates a plot using `matplotlib`, and saves the
results back to OMERO.

For the full script go directly to :doc:`Step 8 <simple_frap_8>`.

**Resources:**
--------------

-  FRAP images available at https://downloads.openmicroscopy.org/images/DV/will/FRAP/

An improved version of this script is also available as `OMERO.script <https://docs.openmicroscopy.org/omero/latest/developers/scripts/index.html>`_:

   -  https://github.com/ome/training-scripts/blob/v0.6.0/practical/python/server/simple_frap.py

   -  https://github.com/ome/training-scripts/blob/v0.6.0/practical/python/server/simple_frap_with_figure.py
